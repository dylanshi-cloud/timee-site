参与 Timee：注册名预留 · 提案 · Commons

本页只讲“怎么参与”。想看“为什么”：去 文明。想看制度规则：去 宪章。

当前阶段的重要边界：不做实名认证发币、不提供法币兑换、不运营投机交易所。我们先把规则、安全和共建机制建立起来。

参与路径（从轻到深）

轻参与：理解与提问

从 FAQ 开始，提出你最担心的风险与反对意见。

去 FAQ →

中参与：写一份提案

公共池项目、规则修改、风控与反诈、应用场景都可以提。

去提案模板 →

深参与：贡献 Commons

上传对社会有建设性的知识、资源、经验，成为公共池与提案的输入。

去 Commons →

1) 预留账户名（仅意向登记，不产生任何 TEE）

你可以先提交意向账户名，帮助我们未来设计“一人一账户”的身份体系。注意：这不是正式开户，不产生任何 TEE，不收集证件信息。

  • 打开 GitHub Issues： https://github.com/dylanshi-cloud/timee-site/issues
  • 新建 Issue，标题：账户名预留：your_desired_name
  • 正文建议写：你想预留的名字 + 你想参与的方向 + 你最担心的风险（可选）。

2) 提案与治理（把讨论写成可执行文本）

  • 提案写作入口：提案模板
  • 提交方式（当前）：GitHub Issues，标题建议：[提案] 简短标题
  • 最重要的一条：请务必写出“风险与失败场景”，这会显著提高提案质量。

3) Commons（把建设性贡献变成文明的输入）

Commons 是“共享的知识与资源广场”。它让人们不只是围绕制度争论，还能持续贡献对他人有用的东西。

  • 入口：Commons 页面
  • 提交方式（当前）：GitHub Issues，标题建议：[Commons] 简短标题
  • 未来：AI 会帮助整理结构、打标签、提示滥用风险，并把 Commons 推荐给相关提案。
